Data Visualization with C3.js

This visualization shows the venture capital industry in the past 20 years. By analyzing the trend in number of deals, breakdown of deals by stage, and pre-money valuation of companies, we are able to relate the growth of tech industry with what we observe in the VC industry. The visualization comes in the form of slides that allows the user to learn the story step by step. The library used here is C3.js.

Data Visualization with D3.js

This visualization shows a baking recipe by plotting it into a tree structure, demonstrating each different pastries and its ingredients. The transition effect on size and color of the nodes indicate the number of children each node has. The user can interact with the graph by folding the nodes, filtering according to certain criteria and drag or zoom the whole chart. It also features an "auto-draw" function that grows the tree from root node by node. The library used is D3.js.
